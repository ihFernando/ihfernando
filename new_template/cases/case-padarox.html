Projeto

O La Padarox, foi um sistema desenvolvido como TCC na Etec de Itanhaém, onde tinhamos como objetivo, realizar o primeiro sistema focado para entregas de produtos para padarias da região. O sistema teve duas áreas, uma para o cliente e outra para o administrador.

Problema

Baseando-se nos estudos de caso, nós tinhamos que montar um site com uma navegação simples e rápida, visando a qualidade e efeciência que nós propusemos a entregar. A ideia, era de que não importasse o dispositivo que o acessasse, ele teria um layout intuitivo e leve.

Solução

Desenvolvemos um sistema com duas áreas, uma para o cliente e outra para o administrador. Fiquei reponsável por elaborar ambas as páginas (layout) e escrever o código.

Tecnologias utilizadas: Bootstrap, Jquery, Photoshop, PHP e MySQL
